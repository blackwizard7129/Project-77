Project on console

To set up console use the following steps: 1)Download visual studio for your operating system. 2)Create a new folder in dektop. 3)Go to visual studio, file, open folder. 4)Then in file, click on open file and type the name of the file.

console screen is used to tell the viewers something which can make sense or can't make sense. You need to right click on the screen and click on inspect then console to open the console. You need to use the console tag (console.log) to print anything.
